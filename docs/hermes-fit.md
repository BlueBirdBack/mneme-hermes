# How Mneme Ideas Fit Hermes

## Existing Hermes strengths
Hermes already has:
- built-in curated memory
- external memory-provider architecture
- session search
- wiki-style knowledge compilation
- cron automation
- plugin and skill extension systems

## Where Mneme can help
Mneme-like ideas fit best in these gaps:
- memory health checks
- provenance-preserving promotion of facts
- contradiction and stale-fact detection
- repo/git delta tracking for wiki maintenance
- structured compiled memory packs derived from noisier evidence

## Most promising initial integration
The best first implementation is probably not a heavyweight plugin.

Instead:
- start as scripts plus docs/specs
- pair with a Hermes skill or wiki workflow
- later promote into a plugin only if runtime integration becomes clearly valuable
